> My utopian vision for how eventing can be done differently.


## Install

```sh
$ npm install --save voltrevo-event-emitter
```


## Usage

```js
var voltrevoEventEmitter = require('voltrevo-event-emitter');

voltrevoEventEmitter('Rainbow');
```
